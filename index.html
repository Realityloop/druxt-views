DruxtJS Views

Drupal Views Vue.js components for Nuxt.js

Get started

The DruxtJS Views module adds Drupal (opens new window) Views support to your NuxtJS (opens new window) application.

# The DruxtView component

Druxt Views provides a Vue.js component to render a Drupal View.

<DruxtView :display-id="displayId" :uuid="uuid" :view-id="viewId" />
1

Get started with the Guide and API Documentation.

# Druxt component

The DruxtEntity component is compatible with the DruxtJS component theming system:

<Druxt module="view" :props-data="{ displayId, uuid, viewId }" :wrapper="wrapper">
1

See the Druxt component documentation (opens new window) for more information.

# DruxtJS

DruxtJS is a suite of modules to connect a Drupal (opens new window) JSON:API backend to a Nuxt.js (opens new window) frontend.

Find out more at http://druxtjs.org (opens new window)
