# .displayId : string

The View Display ID.

Kind: static property of props
Default: "default"

# .type : string

JSON:API Resource type.

Kind: static property of props
Default: "view--view"

# .uuid : string

The View UUID.

Kind: static property of props

# .viewId : string

The View ID.

Kind: static property of props

IDs of displays to be attached after the view.

Kind: static property of computed

# .attachments_before : Array.<string>

Kind: static property of computed
